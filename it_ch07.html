① 半導体メモリ

半導体とは、電気を通す「導体」と通さない「絶縁体」の中間の性質を持つ物質です。温度変化や不純物で電気抵抗が変わります。半導体メモリは、この性質を使ってデータを保存する装置です。

RAMとROMの違い

RAM（Random Access Memory）

揮発性メモリ
電源を切るとデータが消える
用途：主記憶装置、キャッシュメモリ

ROM（Read Only Memory）

不揮発性メモリ
電源を切ってもデータが残る
用途：SSD、USBメモリ

RAMの種類：DRAMとSRAM

DRAM（Dynamic RAM）
・絶えずリフレッシュ（電荷の再書込み）が必要
・回路が単純 → 大容量を安く作れる
・主に主記憶装置（メインメモリ）で使用
・SDRAM（Synchronous DRAM）：クロックと同期して動作するDRAM
SRAM（Static RAM）
・リフレッシュ不要 → 非常に高速
・値段が高い
・主にキャッシュメモリで使用

DRAM

リフレッシュ：必要
速度：中速
価格：安い（大容量向き）
用途：主記憶装置

SRAM

リフレッシュ：不要
速度：高速
価格：高い
用途：キャッシュメモリ

フラッシュメモリ

フラッシュメモリ：電気的にデータの書き換えができるROM（不揮発性メモリ）

・SSD、USBメモリの中身はすべてフラッシュメモリ
・データの読み書き：ページ単位
・データの消去：ブロック単位（ページより大きい単位）

試験キーワード：「ブロック単位で電気的にデータを消去」→ フラッシュメモリ

② メモリセルと電子部品

メモリセルとは

メモリセルとは、半導体メモリの中で「0」か「1」を1ビット保存する最小単位の入れ物です。半導体メモリは無数のメモリセルで構成されています。

DRAMとSRAMの内部構造

DRAM → コンデンサ

コンデンサ：電荷を蓄える電子部品

・電荷の有無で0/1を記憶
・電荷はすぐ放電 → リフレッシュが必要
・構造が単純 → 高集積化・低コスト

SRAM → フリップフロップ回路

フリップフロップ回路：2つの安定状態を持つ順序回路

・電流が流れ続ける限り状態を保持
・リフレッシュ不要 → 高速
・構造が複雑 → 高コスト・低集積
試験での出題パターン：
・「コンデンサに蓄えた電荷の有無で記憶」→ DRAM（またはSDRAM）
・「フリップフロップ回路を利用」→ SRAM
・「二つの安定状態をもつ順序回路」→ フリップフロップ回路

その他のハードウェア用語

チャタリング：機械式スイッチのON/OFFが切り替わる短い間に、意図しないON/OFFが繰り返される現象
例：マウスを1回クリックしたのにダブルクリックと認識される
7セグメントLED：7つのLED（＋小数点用1つ = 計8個）を組み合わせて数字を表示するディスプレイ

・アノードコモン型：低い電圧で光る
・カソードコモン型：高い電圧で光る
第7章の最重要まとめ

・RAM＝揮発性（電源OFF→消える）、ROM＝不揮発性（残る）
・DRAM＝リフレッシュ必要・安い → 主記憶
・SRAM＝リフレッシュ不要・高速 → キャッシュ
・フラッシュメモリ＝ROM、ブロック単位消去 → SSD/USBメモリ
・DRAM → コンデンサ（電荷の有無）
・SRAM → フリップフロップ回路（2つの安定状態）
